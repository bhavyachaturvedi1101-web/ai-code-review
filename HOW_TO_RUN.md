# How to Run

## Step 1 - Open VS Code terminal and run backend
```
mvn spring-boot:run -f "C:\Users\bhavu\Ai code\ai-code-review\backend\pom.xml"
```

## Step 2 - Open a second terminal and run frontend
```
cd "C:\Users\bhavu\Ai code\ai-code-review\frontend"; npm run dev
```

## Step 3 - Open browser
```
http://localhost:5173
```

Keep both terminals open while using the app.
